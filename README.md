# HTMLang

HTML as a Programming Language. Finally all of your dreams came true.

## Hello, World

```html
<!DOCTYPE html>
<html>
  <head>
    <title>HelloWorld</title>
  </head>
  <body>
    <htmlang>
      <call target="console.log"><s>Hello, World</s></call>
    </htmlang>
    <script src="./HTMLang.js"></script>
  </body>
</html>
```
